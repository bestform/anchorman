anchorman by bestform

A plugin for sublime text 2 for creating links and lists of links.

i use this plugin to create the shownotes for the Datenschorle podcast.

It's currently got three functions:

Get Link

Mark some text and anchorman will make a link with the first search result using the marked text as query.

Get Link (multiple choice)

Same as "Get Link", but anchorman will present you with a list of results from which you can choose one.

Make Link from URL

Mark an URL and anchoman will try to find out the title of the page an create the link
The plugin also comes with a handy snippet, triggered by <a[TAB]. Try it out.

Dependencies

Sublime Text 2

Install

For the search function to work, you'll have to provide your own bing (yes, THAT bing, don't ask) API key in an anchorman.sublime-settings file in your user dir.
